# install

copy the "Canduino" folder into the "hardware" subfolder of the Arduino sketchbook location. 
The Arduino sketchbook location is typically in your Documents folder in a subfolder named "Arduino". 
You can see the sketchbook location in the Arduino IDE Preferences dialog, which is available from the File menu. 

# version
supports the 1.0.x, 1.5.x, and 1.6.x, versions of the Arduino IDE.
